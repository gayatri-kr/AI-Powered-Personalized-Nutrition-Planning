Deep learning fusion model for personalized nutrition prediction. Combines health metrics, mental wellbeing data, genomic SNPs, and protein features using LSTM + Transformers. Trained on 73k+ NHANES records and 4.9k genomic profiles to predict key nutrient requirements.

1. Built a multi-branch fusion model integrating health metrics, mental-wellbeing scores, genomic SNP sequences, and protein-folding data using LSTM + Transformer architectures to predict personalized nutrient requirements.
2. Merged and preprocessed 73,778 dietary records from NHANES and 4,983 genomic profiles from the 1000 Genomes Project, handling missing values, scaling, data alignment, and multi-format feature construction.
3. Designed an optimized multi-output regression system predicting grams, energy, protein, carbs, sugars, fiber, fat, and cholesterol with strong model convergence, statistical validation, and deployment-ready architecture.
